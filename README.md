Hashcat Pool
=========

Hashcat Pool is a javascript-based server which decrypts hashes using CPUs of client logged.

Current version is **0.13**

### Currently supported hash type:
- [    0 ] MD5
- [   11 ] Joomla
- [  100 ] SHA1
- [ 1400 ] SHA256
- [ 1700 ] SHA512
